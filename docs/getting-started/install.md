# 安装与准备

本页说明安装 APK 前后的必要准备，包括设备要求、系统开关、ROM 兼容设置和电池策略。

## 适用设备

- Android 11（API 30）及以上真机。
- 无 Root 设备需要开启无线调试，用于完成 Wireless ADB 启动。
- Root 设备需要能正常授予 `su` 权限。

## 安装 APK

1. 从 GitHub Releases 下载 APK。
2. 在手机上安装 APK。
3. 第一次打开 AutoLXB 时，按应用引导完成必要权限设置。

## 必须设置

开始启动 Core 前，请先确认下面几项：

- 已启用 **开发者选项**。
- 已打开 **USB 调试**。
- 无 Root 设备已打开 **无线调试**。
- 根据你的 ROM 完成额外兼容设置：

| ROM | 建议操作 |
| --- | --- |
| MIUI / HyperOS | 开启 **USB 调试（安全设置）** |
| ColorOS | 关闭 **权限监控** |
| Flyme | 关闭 **Flyme 支付保护** |

!!! warning "USB 调试需要保持开启"
    AutoLXB 的设备端服务依赖调试环境保活。即使已经完成配对，也建议保持 USB 调试开启，否则关闭前台应用后可能需要重新启动 Core。

## 电池策略

建议把 AutoLXB 的电池策略设置为 **无限制**，避免配对或任务执行期间服务被系统杀掉。
